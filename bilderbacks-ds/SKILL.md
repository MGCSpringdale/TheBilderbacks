## Quick reference
- **Tokens / global CSS:** `styles.css` (links everything). Color, type, spacing, effects under `tokens/`.
- **Palette:** warm earth — cream/oat surfaces, walnut text & dark sections, **rust** primary, **denim** secondary, **gold** for scripture.
- **Type:** Newsreader (display serif + scripture, italic), Hanken Grotesk (body/UI), Caveat (rare script flourish). Fonts load from Google Fonts CDN.
- **Components:** `components/core/` — Button, Badge, Card, Input, Avatar, ScriptureBlock, SectionHeading. Read each `*.prompt.md`.
- **UI kit:** `ui_kits/website/` — full interactive band site (Home, Music, Tour, About).
- **Voice:** warm, sincere, family-first; "we/our family" + "you"; sentence case; scripture in italic serif. No emoji, no hype. See README "Content fundamentals".
- **Facts:** Southern Gospel family ministry, Springdale AR, since 1997 (29 years). Members: Euel, Marsha, Blake, Quincie, Cortni & Caleb. Albums incl. Full Circle, Our Family Heritage. Booking 479-200-3599 / ebilderback@hotmail.com.
- **Feel:** modern-clean contemporary worship, soft generous radii, warm diffuse shadows, full-bleed natural-light photography with walnut scrims. See README "Visual foundations".
